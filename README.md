# Algorithms

This UI application shows 7 different algorithms that can be run given user input. These algorithms are:
- Encryption
- Fibonacci numbers
- Sorting algorithms (bubble and selection sort)
- Brute force (divide and conquer)
- Randomisation
- Recursion (Factorial numbers)
- Search (Largest, smallest, mode and median)

# Requirements:
Python 3.10+  
Tkinter  
Functools

# Running
To run this program, ensure that all files are in one directory, then run main.py
